# QC Teazzi

Aplikasi mobile untuk proses QC otomatis dengan template Excel yang dapat disesuaikan.

## Fitur

- 📸 Pilih foto QC dari galeri
- 📊 Masukkan ke template Excel otomatis
- 📱 Kompatibel Android API 24+
- 🌐 Download template dari GitHub
- 📤 Share hasil langsung
- 📊 Support LGJA, Sultan, dan Vano layouts

## Setup & Build

Silakan lihat dokumentasi build di direktori root atau di file ini untuk instruksi lengkap setup development.

## struktur Folder

```
qc-teazzi/
├── core/                 # Core modules
├── screens/              # UI Screens
├── kv/                   # Kivy layout files
├── assets/               # Icons & images
├── main.py              # Entry point
├── buildozer.spec       # Build config
└── requirements.txt     # Python dependencies
```
